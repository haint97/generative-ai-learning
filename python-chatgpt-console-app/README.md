To run your ChatGPT console app:

1. Open a terminal and navigate to the project folder:
   ```
   cd python-chatgpt-console-app
   ```

2. (Optional) Create and activate a virtual environment if not already done:
   ```
   python3 -m venv .venv
   source .venv/bin/activate
   ```

3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

4. Set your OpenAI API key (recommended) in your terminal:
   ```
   export OPENAI_API_KEY=
   ```

5. Run the chat app:
   ```
   python src/main.py
   ```

You can now chat with ChatGPT in your terminal. Type exit to quit.
